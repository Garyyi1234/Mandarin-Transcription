Mandarin captioning in camera to help my grandfather (hard of hearing) understand broken mandarin more clearly.

Installation:
1. Clone the repo
2. Run setup_driver.bat
3. This should create a folder "UnityCapture-master"
4. Run Install.bat in "UnityCapture-master/Install" as administrator to register the virtual camera
5. Paste api keys in .env_format and rename to .env

Run:

6. pip install -r requirements.txt

7. python main.py

Optionally, run: python main.py 1
To mirror text left-right if your meeting software automatically mirrors camera output
